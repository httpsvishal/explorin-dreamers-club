Introduction to HTML

In this page we are going to learn Html and its basic tags

HTML, or HyperText Markup Language, is the standard markup language used to create and design web pages. It provides the structure for content on the web, using a series of elements and tags to define the various parts of a webpage.

The < head > element in HTML contains meta-information about the document, which is not displayed directly on the webpage. Instead, it provides information to the browser, search engines, and other web services.

elements in head tag

  1. < meta charset="UTF-8" >: Declares the character encoding for the document. UTF-8 is a widely used encoding that supports a broad range of characters.
  2. < meta name="viewport" content="width=device-width, initial-scale=1.0">: Defines the viewport settings, helping to ensure that the webpage is displayed properly on various devices and screen sizes. The width=device-width ensures that the width of the viewport is set to the device's screen width, and initial-scale=1.0 sets the initial zoom level to 1.
  3. < title>My Web Page< /title>: Sets the title of the HTML document, which is displayed in the browser's title bar or tab. It's also used by search engines.

Additional elements

  • Linked Stylesheets: To apply external styles to the document.ex- < link rel="stylesheet" type="text/css" href="styles.css">
  • JavaScript Scripts: To include external scripts or link to scripts.ex-< script src="script.js">
  • Favicons: Small icons displayed in the browser tab or address bar. ex-< link rel="icon" type="image/png" href="favicon.png">
  • Meta Tags for SEO: Providing information to search engines. ex- < meta name="description" content="Description of the webpage">
    < meta name="keywords" content="keywords, separated, by, commas">
